# 🖼️ 网页图片下载器 — Chrome 扩展

一键提取、筛选、批量下载当前网页上的所有图片。支持懒加载图片、CSS 背景图、防盗链绕过、独立灯箱预览（含缩放拖拽）。

## ✨ 功能总览

| 功能 | 说明 |
|------|------|
| 🔍 **一键提取** | 提取页面上所有 `<img>`、CSS `background-image`、`<picture>`、懒加载 `data-src` 等来源的图片 |
| 🎨 **格式筛选** | 按 JPEG / PNG / GIF / WebP / SVG / AVIF 格式过滤 |
| 📐 **尺寸筛选** | 按最小宽高过滤 |
| 🖼 **网格预览** | 缩略图网格展示，每张标注尺寸和格式，加载失败显示渐变彩色占位符 |
| ✅ **批量勾选** | 全选 / 取消 / 逐张勾选 |
| 📦 **ZIP 打包** | **所有选中图片打包成一个 ZIP 文件，一次下载**，告别每次弹框确认 |
| 🔒 **防盗链绕过** | 通过 `declarativeNetRequest` 动态注入页面 Referer 头，绕过 CDN 防盗链 |
| 🔄 **实时刷新** | 滚动懒加载后点刷新按钮，重新扫描页面图片 |
| 🖥 **独立灯箱预览** | **双击图片弹出独立全屏窗口**，支持以下功能 |
| 🔍 **缩放拖拽** | **Ctrl + 滚轮** 缩放（25% ~ 1000%），**拖拽**平移，**双击**还原 |
| ⬇ **单张下载** | 预览窗口内随时下载当前图片，也支持在新标签页打开 |

## 📦 安装

### 从源码加载（开发者模式）

```bash
# 进入项目目录
cd chrome-image-downloader
```

1. 打开 Chrome → `chrome://extensions`
2. 开启右上角 **"开发者模式"**
3. 点击 **"加载已解压的扩展程序"**
4. 选择 `chrome-image-downloader/` 目录
5. Chrome 会弹出权限确认 → 点击 **"扩展程序"** 或确认

## 🎯 使用指南

### 基本流程

1. 访问任意包含图片的网页（如 X/Twitter、微博、知乎、电商网站等）
2. 点击扩展图标 ![](icons/icon16.png)
3. 等待自动扫描，弹出面板显示所有图片
4. 在工具栏筛选格式 / 最小尺寸
5. 勾选需要的图片（单击切换选中）
6. 点击 **"下载选中"** → 浏览器下载一个 ZIP 包
7. 解压 ZIP → 得到所有原图（保留原始文件名）

## 🔐 权限说明

| 权限 | 用途 |
|------|------|
| `activeTab` | 获取当前标签页 URL，注入提取脚本 |
| `scripting` | 在页面中注入图片提取脚本 |
| `downloads` | 下载打包好的 ZIP 文件 |
| `declarativeNetRequest` | 动态注入 Referer 请求头（绕过 CDN 防盗链） |
| `host_permissions: \<all_urls\>` | 后台 Service Worker 跨域抓取原图 |

> ⚠️ `host_permissions` 仅用于后台脚本下载跨域图片，**不收集、不发送任何用户数据**。
> 所有代码运行在本地，无外部请求（除了抓取你指定的图片本身）。


### 核心流程

```
用户点击 "下载选中"
  │
  ├─ ① popup 提取选中图片 URL + 当前页面 URL（作为 Referer）
  ├─ ② 通知后台 SW 创建 DNR 规则（注入 Referer 头）
  ├─ ③ 后台 SW 用 fetch() 逐张抓取原图
  │     └─ host_permissions 免除 CORS 限制
  │     └─ DNR 规则自动带上 Referer → CDN 放行原图
  ├─ ④ 后台 SW 用纯 JS 打包所有图片为 ZIP 文件
  ├─ ⑤ chrome.downloads.download 下载 ZIP 到本地
  └─ ⑥ DNR 规则自动清理
```

### 防盗链原理

许多图片 CDN（如 `pbs.twimg.com`）检查 HTTP `Referer` 头来防止盗链。

- **右键另存为**：浏览器自动带上页面 Referer → CDN 放行 → 原图
- **`chrome.downloads.download`**：**不携带页面 Referer** → CDN 返回缩略图或错误页（例如 60KB → 7KB）

本扩展通过 `declarativeNetRequest` API 动态创建规则，**每张图片下载前都临时注入正确的 `Referer` 头**，下载完成后自动清理规则。

### ZIP 打包（纯 JS 实现）

为了一次性下载多张图片且不触发 Chrome 的多重下载保护弹窗，后台 Service Worker 在内存中用纯 JavaScript 构建 ZIP 文件：

- **STORE 模式**（不压缩，速度最快）
- 无需任何外部依赖库
- 支持任意数量图片（受浏览器内存限制，建议单次不超过 100 张）
- 文件中保留原始文件名
